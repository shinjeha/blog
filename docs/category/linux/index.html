Temps de lecture: 1 min

.bashrc쉘에 등록

alias는 터미널을 껐다가 키면 사라지는데 영구히 저장하고 싶다면 .bashrc쉘에 등록해줘야 한다.

# home디렉토리의 .bashrc쉘 열기($HOME/.bashrc)
# 난 root계정으로 실행했으므로
vim /root/.bashrc

# 빈 자리에다가 alias등록
alias ll='ls -al'
